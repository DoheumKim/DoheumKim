### 🧐 **Project Experience**

- 👟 **Allbirds E-commerce Platform (Backend Lead & Frontend)**
  - **Description**: 친환경 신발 브랜드 'Allbirds'의 쇼핑몰 클론 프로젝트 (기여도: Backend 100%, Frontend 'My Page' 담당)
  - **Github**: [DoheumKim/Allbirds-CloneCoding](https://github.com/DoheumKim/Allbirds-CloneCoding)
  - **Tech Stack**: Node.js, Express, MongoDB, React
  - **Key Responsibilities**:
    - **Robust API Architecture**: 사용자(Auth), 상품(Product), 장바구니(Cart), 주문(Order) 등 **5개 핵심 도메인에 대한 RESTful API** 설계 및 구현
    - **Frontend Integration (My Page)**: 주문 내역(Order History)을 시각화하는 **마이페이지 UI/UX 직접 구현** (React, Axios)
    - **Advanced Search & Filter**: 쿼리 스트링을 활용한 다중 필터링(가격/사이즈/재질) 및 정렬(최신순/리뷰순) 로직 개발
    - **Admin System**: 상품 등록/수정 및 기간별 판매 현황(Sales)을 조회하는 **관리자 전용 대시보드 API** 개발
    - **Security & Auth**: `bcrypt`를 이용한 암호화 및 `express-session` + `connect-mongo` 기반의 세션 인증 구현
    - **Database Modeling**: Mongoose를 활용하여 상품 옵션(사이즈, 재질)과 주문 내역을 효율적으로 관리하는 스키마 설계
